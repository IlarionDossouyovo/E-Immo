# E-Immo Platform

Plateforme immobilière multi-entreprises au Benoît.

## Pages

- **index.html** - Page d'accueil de la plateforme
- **dashboard.html** - Tableau de bord entreprise
- **listings.html** - Liste de toutes les annonces
- **company/electron.html** - Page entreprise ELECTRON
- **property/detail.html** - Détail d'une propriété
- **404.html** - Page d'erreur

## Structure

```
E-Immo/
├── css/
│   └── styles.css
├── js/
│   └── platform.js
├── company/
│   └── electron.html
├── property/
│   └── detail.html
├── index.html
├── dashboard.html
├── listings.html
├── 404.html
├── robots.txt
└── sitemap.xml
```

## Utilisation

1. Ouvrir `index.html` dans un navigateur
2. S'inscrire comme entreprise
3. Ajouter des-annonces
4. Les clients peuvent voir et payer
